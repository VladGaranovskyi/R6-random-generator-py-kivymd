# R6Random
This app allows you to generate random setups for custom games in Tom Clancy's Rainbow Six Siege game.

Features:
- `Random team`;
- `Random attack operator`;
- `Random defense operator`;
- `Random map`;
- `Random settings`;
- `Random mini game`;
- `Random number of matches`;
- `Random rank`;
